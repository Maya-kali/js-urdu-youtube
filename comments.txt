ctrl + space = show suggestion
ctrl + / = comments
ctrl + shift + L = multi cursor selection
ctrl + p display the search bar
ctrl + B = hide and show side bar
alt + up and alt + down = move line
ctrl + o = open current file
ctrl + d = one by one selection
ctrl + shift + E = move to explore window 
ctrl + x = delete line
ctrl + c = copy / ctrl + v = paste
ctrl + c + downArrow = copy 

ctrl + shift + [ or ] = fold or unfold
